# Ascension 100

Extends Slay the Spire's Ascension mode from level 20 to level 100.

## Features

- **100 Ascension Levels**: Progressive difficulty increase from level 1 to 100
- **Multilingual Support**: Full Korean and English localization
- **Mod Compatibility**: Compatible with base game characters and other mods
- **Fine-tuned Difficulty**: Unique modifiers for each level

# 승천 100

Slay the Spire의 승천 모드를 20레벨에서 100레벨까지 확장하는 모드입니다.

## 기능

- **100개 승천 레벨**: 레벨 1부터 100까지 점진적 난이도 증가
- **다국어 지원**: 한국어, 영어 완벽 지원
- **기존 모드 호환**: 베이스 게임 캐릭터 및 다른 모드와 호환
- **세밀한 난이도 조정**: 각 레벨마다 고유한 수정자 추가
